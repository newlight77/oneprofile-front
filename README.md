## Requirements

- [Nodejs 8+ and Npm 6+](https://nodejs.org/en/download/)

```sh
npm install -g @angular/cli
```

## bootstrap using @angular/cli

```sh
ng new oneprofile-front
```

> Angular routing : yes
> stylesheet : SCSS
